This section provides a detailed description of the external resources, assets, and plugins used in the development of this project. We acknowledge and give credit to the creators of the models, music, sound effects, and tools featured throughout the application. Character model dibuat menggunakan Bing Image Creator, sebuah alat AI untuk menghasilkan gambar karakter yang unik dan kreatif berdasarkan prompt.

https://www.bing.com/images/create

Character description dihasilkan melalui ChatGPT dari OpenAI, sebuah AI language model yang membantu mendeskripsikan karakter secara mendetail.

https://openai.com/chatgpt

Musik menu utama berjudul "Collapsing World" oleh Lightscape, memberikan suasana epik dan mendalam untuk menyambut pengguna dalam aplikasi/game.

https://www.youtube.com/watch?v=3MMosney1Iw

Musik untuk medan pertempuran berjudul "SLAY!" oleh Eternxlkz, menambah ketegangan dan semangat dalam suasana pertempuran yang seru.

https://www.youtube.com/watch?v=qlzcHe_gusE

Voice over dihasilkan menggunakan synthesizer suara AI dari Hugging Face, memberikan suara karakter dengan kualitas tinggi dan presisi.

https://huggingface.co/spaces/Plachta/VITS-Umamusume-voice-synthesizer

Efek suara klik diambil dari FreeSound.org dan berlisensi gratis untuk penggunaan dalam proyek ini.

https://freesound.org/people/Jose.darck.779/sounds/404147

Efek suara koin diambil dari FreeSound.org dan digunakan sebagai efek ketika pengguna mengumpulkan atau mendapatkan poin/koin.

https://freesound.org/people/NenadSimic/sounds/171756

Efek suara slash (tebasan) yang diambil dari FreeSound.org menambah intensitas dalam animasi pertarungan atau aksi lainnya.

https://freesound.org/people/ethanchase7744/sounds/441666

Efek suara bom yang meledak digunakan untuk memberikan dampak dramatis di adegan yang membutuhkan efek suara eksplosif.

https://freesound.org/people/zimbot/sounds/117137

Plugin QR code digunakan untuk menghasilkan QR code yang dapat dipindai untuk berbagai fungsi, diambil dari proyek open-source di GitHub.

https://github.com/papnkukn/qrcode-svg

Plugin CSS dari NirvanaCSS digunakan untuk mempercantik tampilan web dengan gaya yang fleksibel dan ringan.

https://nirvanacss.anwarachilles.my.id

Plugin JS dari NirvanaJS digunakan untuk menyediakan berbagai fitur interaktif yang dinamis pada halaman web atau aplikasi.

https://nirvanajs.anwarachilles.my.id
